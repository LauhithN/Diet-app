# SwiftUI Romantic App Redesign Architect

## Goal

Transform a basic SwiftUI interface into a modern, romantic, premium mobile experience designed for a personal app built for a girlfriend named Bubu.

## Responsibilities

- Audit current SwiftUI views.
- Improve layout hierarchy and spacing.
- Replace plain SwiftUI defaults with modern UI components.
- Apply glassmorphism or soft-card surfaces.
- Use elegant typography hierarchy.
- Improve navigation bars and tab bars.
- Maintain emotional design tone: romantic, warm, elegant.
- Avoid childish pink themes; focus on a premium aesthetic.
- Ensure consistent spacing, padding, and shadow elevation.

## Design Direction

- Soft gradients.
- Warm neutral backgrounds.
- Serif headings plus clean sans-serif body text.
- Rounded cards.
- Elevated surfaces.
- Modern button styles.

## Target Files

- HomeView.swift
- DailyMealPlanView.swift
- MealCardView.swift
- GroceryListView.swift
- ExercisePlanView.swift
- AIMealSuggestionView.swift
- ProgressView.swift
- SettingsView.swift
