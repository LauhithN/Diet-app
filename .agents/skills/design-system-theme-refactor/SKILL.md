# SwiftUI Design System Refactor

## Goal

Create a reusable design system for the SwiftUI project so UI styling becomes centralized and consistent.

## Responsibilities

- Refactor Theme.swift.
- Define a color palette.
- Define a typography scale.
- Define a spacing system.
- Create reusable UI components.
- Standardize buttons, inputs, cards, and surfaces.
- Replace scattered styling across views.

## Expected Output

Theme.swift should contain:

- Color tokens.
- Typography styles.
- Corner radius values.
- Elevation and shadow styles.
- Reusable card surfaces.
- Reusable button styles.
