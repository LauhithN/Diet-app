# SwiftUI Build and Verification Agent

## Goal

After UI changes, ensure the SwiftUI project compiles successfully.

## Responsibilities

- Run Swift build checks.
- Detect SwiftUI compile errors.
- Fix minor syntax issues.
- Verify previews compile.
- Confirm navigation stack works.
- Ensure tab bar styling works.
- Ensure Theme.swift integration does not break the app.
