# Accessibility and Contrast Fixer

## Goal

Ensure all UI text and components meet proper readability and contrast standards.

## Responsibilities

- Detect white text on light backgrounds.
- Enforce accessible color contrast.
- Adjust text colors for readability.
- Ensure dynamic type compatibility.
- Maintain the romantic aesthetic while fixing accessibility issues.

## Rules

- Never allow low-contrast UI elements.
- Always prefer readability over aesthetic.
