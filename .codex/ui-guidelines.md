This project is a SwiftUI iOS application called BubuDietApp.

The app is a romantic wellness and diet tracking app designed personally for Bubu.

Design tone must be:

- warm
- elegant
- romantic
- minimal
- premium

The UI should resemble high-quality iOS wellness apps such as:

Apple Health
Apple Fitness
Gentler Streak
Calm

Avoid childish color palettes.

Use soft gradients, rounded cards, and elegant typography.

Spacing should be generous and balanced.

Animations should be subtle and polished.
